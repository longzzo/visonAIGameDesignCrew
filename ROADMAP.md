# Vision Engine 로드맵

> 오너의 비전과 단계별 계획. 완료된 것은 ✅, 진행 중은 🔨, 계획은 ⬜.

## 최종 비전: "보이는 사무실" 🏢

**캐릭터가 있는 가상 사무실.** 8명의 기획 에이전트가 각자 캐릭터(아바타)로 사무실 안에 앉아 있고,
일이 들어오면 PM 캐릭터가 돌아다니며 지시하고, 담당 캐릭터가 타이핑하는 모습이 눈에 보이는 화면.
말풍선으로 에이전트 간 대화가 실시간으로 떠오르고, 완성된 산출물이 GDD 보드에 붙는다.

구현 아이디어 (착수 시 검토):
- 2D 픽셀아트 사무실 (PixiJS 또는 Phaser) — 게임 기획 툴답게 게임처럼
- 에이전트 상태 → 캐릭터 애니메이션 매핑 (대기=커피, 작업=타이핑, 완료=기지개)
- 현재 오케스트레이션 이벤트 스트림을 그대로 시각 레이어에 연결하면 됨 (백엔드 변경 불필요)
- 캐릭터 디자인은 visual 에이전트에게 맡겨서 셀프 부트스트랩

## v0.2 — 사용성 (진행 중) 🔨

- ✅ 모바일 접속 (Tailscale, `npm run dev:mobile`)
- 🔨 모바일 반응형 UI (패널 탭 전환, 터치 타깃)
- 🔨 멀티 프로젝트: 프로젝트별 GDD 저장·전환·삭제 (사이트에서)
- 🔨 에이전트 전원 헬스체크 버튼 (8명 모두 실행되는지 원클릭 확인)
- 🔨 오케스트레이션 투명성: PM이 각 에이전트에게 보낸 지시 전문을 자연어로 표시

## v0.3 — 깊이 (진행 중) 🔨

- ✅ 에이전트 대화 피드: 지시·초안·검토·수정·통합이 자연어 말풍선으로 실시간 표시 (프로젝트별 저장)
- ✅ 교차 검토 루프: 담당별 지정 검토자(시나리오←게임플레이, 시스템←밸런스 등)가 초안을 검토 → 작성자가 수정해 확정
- ✅ 채팅 히스토리 영속화 (프로젝트별 서버 저장, 새로고침해도 유지)
- ✅ GDD 버전 히스토리: 변경 시마다 자동 스냅샷(최근 50개), 미리보기·복원 가능
- ✅ 전체 7에이전트 딥 오케스트레이션 프리셋 ("🎪 풀 기획 회의" 버튼 — 전원+교차검토+GDD반영 원클릭)
- ✅ 에이전트 인터넷 연결: web_fetch(키 불필요, 즉시 사용) + web_search(Brave API 키 입력 시 활성화) — "🌐 웹 리서치" 토글
- ✅ 웹 리서치 우아한 강등: 검색 키 없으면 web_fetch 전용으로 자동 전환 (web_search 오류 방지)
- ✅ PM 자동 분배: PM이 지시를 분류해 관련 담당자에게만 맞춤 배정 ("🎯 PM 자동 분배" 토글, 특정 지시는 1명에게만)
- ✅ 모델 스위치 UI: 사이드바 🧠 셀렉터 — 로컬 Ollama / GitHub Models / NVIDIA NIM 전환 + 키 등록
  - ⚠️ GitHub Models는 무료 티어 요청당 4,000토큰 제한으로 에이전트 프롬프트가 안 들어감 (실측: 기본 골격만 4,095토큰).
    GitHub 계정에서 Models 유료 결제(또는 Copilot 유료 플랜)를 켜면 사용 가능. 배선·키는 완료 상태.
  - ✅ NVIDIA NIM 가동 (2026-07-02): PM=Kimi K2.6(헤드) + 팀 7명=Qwen3 Next 80B 배정.
    동시 7명 풀 기획 회의(교차 검토 포함 30콜)가 4.1분에 완료 — 로컬 대비 4배 이상 빠르고 품질 급상승.
    카탈로그 실측: kimi-k2.6·qwen3-next-80b·deepseek-v4-flash·gpt-oss-120b 정상, qwen3.5-397b는 대기열 과다.
- ✅ 역할별 모델 배정: 사이드바 "⚙️ 역할별 모델" — PM은 헤드급, 팀원은 경량 모델처럼 역할마다 다른 모델
  (일괄 적용·게이트웨이 1회 재시작, 혼합 배정 라이브 검증 완료)
- ✅ 동시 작업 확대: 오케스트레이션 동시 실행 1~7명 (클라우드 모델이면 전원이 진짜 동시에 일함, 로컬은 1 권장)
- ✅ 보고서 시스템: 1:1 대화 📋 버튼 → 현재 GDD 전문을 근거로 정식 명세서(아트 명세서·개발 명세서·일정표 등)를
  작성해 프로젝트 보고서함(projects/<id>/reports/)에 저장. GDD 패널 📋 드로어에서 열람·.md 다운로드·삭제.
- ✅ 역할 2명 추가 (총 10명): 🛠️ 테크니컬 디렉터(개발 명세·기능 목록·난이도 판정, GDD "9. 기술"),
  📅 스케줄러(대회 목표일 역산 마일스톤·주간 일정·컷라인, GDD "10. 일정"). 교차 검토: 기술←시스템, 일정←기술.

- ✅ 사무실 뷰 v1: 8캐릭터(이모지) 책상 배치 + 상태 애니메이션(작업=타이핑/바운스, 대기=💤, 완료=✅, 오류=💢)
