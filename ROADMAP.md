# Vision Engine 로드맵

> 오너의 비전과 단계별 계획. 완료된 것은 ✅, 진행 중은 🔨, 계획은 ⬜.

## 최종 비전: "보이는 사무실" 🏢

**캐릭터가 있는 가상 사무실.** 8명의 기획 에이전트가 각자 캐릭터(아바타)로 사무실 안에 앉아 있고,
일이 들어오면 PM 캐릭터가 돌아다니며 지시하고, 담당 캐릭터가 타이핑하는 모습이 눈에 보이는 화면.
말풍선으로 에이전트 간 대화가 실시간으로 떠오르고, 완성된 산출물이 GDD 보드에 붙는다.

구현 아이디어 (착수 시 검토):
- 2D 픽셀아트 사무실 (PixiJS 또는 Phaser) — 게임 기획 툴답게 게임처럼
- 에이전트 상태 → 캐릭터 애니메이션 매핑 (대기=커피, 작업=타이핑, 완료=기지개)
- 현재 오케스트레이션 이벤트 스트림을 그대로 시각 레이어에 연결하면 됨 (백엔드 변경 불필요)
- 캐릭터 디자인은 visual 에이전트에게 맡겨서 셀프 부트스트랩

## v0.2 — 사용성 (진행 중) 🔨

- ✅ 모바일 접속 (Tailscale, `npm run dev:mobile`)
- 🔨 모바일 반응형 UI (패널 탭 전환, 터치 타깃)
- 🔨 멀티 프로젝트: 프로젝트별 GDD 저장·전환·삭제 (사이트에서)
- 🔨 에이전트 전원 헬스체크 버튼 (8명 모두 실행되는지 원클릭 확인)
- 🔨 오케스트레이션 투명성: PM이 각 에이전트에게 보낸 지시 전문을 자연어로 표시

## v0.3 — 깊이 ⬜

- ⬜ 에이전트 간 직접 대화(교차 검토): balance가 systems 산출물을 비판하고 수정 제안하는 루프
- ⬜ 채팅 히스토리 영속화 (프로젝트별, 서버 저장)
- ⬜ GDD 버전 히스토리 / 되돌리기 (git 커밋 자동화)

- ⬜ 산출물 품질 개선: GPT Pro / Claude(Fable 5) 모델 스위치 UI

## v0.4 — 보이는 사무실 (비전 1단계) ⬜

- ⬜ 사무실 뷰: 8캐릭터 배치 + 상태 애니메이션
- ⬜ 지시/보고 말풍선 실시간 표시
- ⬜ 캐릭터 디자인 (visual 에이전트 산출물 기반)
