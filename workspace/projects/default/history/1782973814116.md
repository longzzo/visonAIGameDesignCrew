# 마스터 GDD — (프로젝트명 미정)

> Vision Engine이 관리하는 마스터 게임 디자인 문서. PM 디렉터가 각 전문 에이전트의 산출물을 이 문서에 통합한다.

## 1. 개요

**한 줄 피치**: 마법이 스며든 카페에서 세 마리의 고양이 친구들과 함께하는 일상의 작은 모험.  
**장르/플랫폼/타깃**: 퍼즐·스토리 어드벤처 / PC/모바일 / 어린이~성인  
**핵심 루프 3줄**:  
- 고객과의 만남을 통해 카페의 마법을 키우고,  
- 고양이 친구들과의 특별한 순간을 경험하며,  
- 일상 속에서 소소한 위로와 즐거움을 발견한다.  
**차별점 2개**:  
- 마법이 스며든 카페라는 독특한 세계관,  
- 세 마리의 고양이 주인공이 각자 독특한 능력을 지녔다.  
**산출물 간 충돌·조정 필요 사항**:  
- 세계관 설명과 캐릭터 개념이 서로 보완되어야 함,  
- 플레이어 경험과 스토리 전개의 균형 조정 필요.

## 2. 세계관·스토리

고양이 카페 게임은 마법이 스며든 편안한 카페를 배경으로, 세 마리의 특별한 고양이 친구들과 함께하는 일상의 작은 모험을 그립니다.  
카페는 마법의 힘으로 살아 움직이며, 고객들은 고양이들과의 만남을 통해 행복과 희망을 느낍니다.  
세 마리의 주인공 고양이는 각각 독특한 능력을 지녔고, 서로 다른 성격으로 관객들의 마음을 사로잡습니다.  
카페의 마법은 일상 속에서 특별한 순간을 만들어내며, 게임 플레이어는 그 순간들 속에서 소소한 위로와 즐거움을 발견하게 됩니다.  
세상은 평화로우면서도 마법의 잔잔한 흐름 속에서, 고양이 카페는 그 중심이 되며, 모든 이들에게 따뜻한 안식처가 됩니다.

## 3. 게임플레이

플레이어는 실시간으로 이동하고, 주변을 탐색하며 이벤트를 트리거합니다.  
모든 행동은 키 입력으로 제어되며, 캐릭터의 상태와 환경 변화에 따라 다양한 결과가 발생합니다.  
NPC와의 상호작용, 아이템 사용, 퀘스트 수행이 게임의 주요 메커니카입니다.  
턴제가 아닌 연속적인 플레이로, 전략적 판단과 빠른 반응이 중요합니다.  
게임 내에서 수집한 아이템은 강화, 제작, 판매 등 다양한 방식으로 활용됩니다.  
스코어는 경험치와 보유 자원에 따라 결정되며, 최종 목표는 세계를 정복하는 것입니다.  
사용자 인터페이스는 최소한으로 유지되어, 플레이어의 집중력을 해치지 않도록 설계되었습니다.  
게임은 무한 루프로 진행되며, 플레이어의 선택이 세계를 변화시킵니다.  
각 이벤트는 랜덤 요소와 전략적 요소의 조합으로 발생하며, 재미를 극대화합니다.  
플레이어는 자신의 자원을 관리하며, 적을 공략하거나 협상할 수 있습니다.  
게임은 플레이어의 상상력과 전략적 사고를 키우는 데 초점을 맞추고 있습니다.  
각 퀘스트는 독립적인 이야기를 담고 있으며, 완료 시 보상과 새로운 이벤트를 제공합니다.  
플레이어는 자신의 행동에 따라 세계의 균형을 깨거나 복원할 수 있습니다.  
게임은 빠른 피드백을 제공하여, 플레이어가 즉시 자신의 선택의 결과를 확인할 수 있습니다.  
모든 요소는 서로 연결되어 있으며, 플레이어의 선택이 세계의 흐름을 바꿉니다.  
게임은 끝이 없는 재미와 깊은 전략성을 갖춘 코어루프를 제공합니다.

## 4. 시스템

게임은 플레이어의 선택에 따라 세계가 변화하는 실시간 전개형 구조로 설계된다.  
플레이어는 매턴마다 행동을 선택하고, 시스템은 선택에 따라 상태를 업데이트하며,  
이벤트 트리와 랜덤 요소를 결합해 유일한 게임플레이를 생성한다.  
PM은 매턴마다 시스템의 로직과 상태 변경을 검증하며,  
플레이어의 행동이 시스템 규칙과 충돌하는지 점검한다.  
시스템은 모든 상태를 로그로 기록해 PM의 검증을 용이하게 지원한다.  
플레이어의 행동은 텍스트 입력으로 이루어지며, 시스템은 입력을 해석해 행동을 실행한다.  
PM은 시스템 로직이 투명하고 일관되게 작동하는지 확인한다.  
시스템은 PM의 검증 결과에 따라 로직을 수정하거나 추가할 수 있다.  
플레이어는 시스템의 규칙을 존중하며, PM의 검증을 받는다.  
시스템은 PM의 검증을 통해 안정성과 공정성을 확보한다.  
플레이어는 시스템의 상태를 실시간으로 확인할 수 있다.  
시스템은 PM의 검증을 통해 유연한 규칙을 유지한다.  
플레이어는 시스템의 상태 변경을 이해하고, PM의 검증을 협력한다.  
시스템은 PM의 검증을 통해 게임의 흐름을 조절한다.  
PM은 시스템의 로직이 공정하고 투명하게 작동하는지 검증한다.  
시스템은 PM의 검증을 통해 게임의 규칙을 개선한다.

## 5. UI/UX

### 메인 화면
게임 타이틀, 시작 버튼, 설정 버튼, 게임 모드 선택 메뉴를 중앙에 배치.

### 게임 화면
플레이어 캐릭터, 몬스터, UI 패널(HP, MP, 경험치, 아이템)을 포함한 실시간 전투 화면.

### 설정 화면
게임 설정, 음량 조절, 그래픽 설정, 언어 선택, 저장/로딩 옵션을 제공.

### 인벤토리
아이템 목록, 사용/丢弃 버튼, 아이템 설명 패널을 포함한 간단한 인터페이스.

### 대화창
NPC와의 대화창, 선택지 제공, 대화 기록 보관 기능 포함.

### 레벨업 화면
레벨업 시, 스탯 선택 메뉴, 추가 스킬 선택, 경험치 재분배 기능 제공.

### 게임 오버 화면
게임 오버 메시지, 재시작 버튼, 메인 메뉴로 돌아가기 버튼 제공.

### 스토리 화면
스토리 텍스트, 이전/다음 버튼, 스토리 기록 저장 기능 포함.

### 캐릭터 생성 화면
이름 입력, 직업 선택, 외형 설정, 초기 스탯 분배 기능 제공.

### 아이템 설명 패널
아이템 이름, 효과 설명, 사용 방법, 소유자 확인 기능 포함.

### 전투 로그
전투 중 발생한 이벤트, 공격/회피/치명타 기록, 전투 기록 저장 기능 포함.

### 패널 위치
모든 UI 요소는 화면 하단에 고정되어 있으며, 터치/마우스 클릭으로 조작 가능.

### 색상 및 스타일
모던한 2D 스타일, 파스텔 톤, 폰트는 깔끔한 sans-serif 폰트 사용.

## 6. 밸런스

게임 내 모든 유닛, 스킬, 아이템의 강도와 효율을 체계적으로 조절하여 플레이어의 전략적 선택과 균형을 유지합니다.  
각 클래스의 역할과 상호작용을 명확히 하며, 과도한 강화나 약화를 방지해 공정한 경쟁 환경을 제공합니다.  
데이터 기반의 분석과 테스트를 통해 지속적으로 조정하며, 플레이어 피드백을 반영해 게임의 재미와 깊이를 극대화합니다.

## 7. 수익모델

게임 내 인앱 구매를 통해 유료 아이템 판매하며,  
유저의 게임 내 활동 기록을 기반으로 광고를 제공하여 수익 창출.  
또한, 게임 내 콘텐츠 제작자와 협업하여  
커스터마이즈된 콘텐츠를 판매하는 플랫폼 모델 도입.

## 8. 아트

게임의 시각적 스타일은 차분한 색조와 부드러운 라이트 효과를 기반으로 한다. 캐릭터는 2D 스타일로 제작되며, 배경은 평화로운 자연 풍경을 연상시키는 색상으로 구성된다. 애니메이션은 부드러운 트랜지션과 자연스러운 움직임을 강조한다. 아트 자료는 PNG 형식으로 저장되며, 모든 이미지는 1024x768 해상도로 제작된다.
